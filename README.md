#### U : 윗줄 왼쪽으로 한칸 밀기 / U' : 윗줄 오른쪽으로 한칸 밀기
#### R : 오른쪽줄 위쪽으로 한칸 밀기 / R' : 윗줄 아래쪽으로 한칸 밀기
#### L : 왼쪽줄 아래쪽으로 한칸 밀기 / L' : 윗줄 위쪽으로 한칸 밀기
#### B : 아랫줄 오른쪽으로 한칸 밀기 / B' : 윗줄 왼쪽으로 한칸 밀기
